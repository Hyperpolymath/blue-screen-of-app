= Blue Screen of App
:toc:
:icons: font

[.lead]
_Does exactly what it says on the tin... or screen._

A humorous web app displaying authentic Blue Screen of Death errors. Built with *Deno*, *ReScript*, and *Podman*.

== Quick Start

[source,bash]
----
# Install: Deno, Podman, Just (manual install)
just build              # Compile ReScript + export config
just podman-run         # Deploy container
# Visit http://localhost:8080 (or https://yourserver.com via proxy)
----

== Architecture

* *Runtime*: Deno (no Node.js, no npm)
* *Business Logic*: ReScript (type-safe)
* *Server*: Minimal JavaScript glue code
* *Containers*: Podman (rootless)
* *Protocol*: HTTP/1.1 (use nginx/caddy for HTTPS)
* *Config*: Nickel ‚Üí JSON export
* *Port*: 8080 (proxied to 443)
